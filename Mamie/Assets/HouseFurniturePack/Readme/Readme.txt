HDRP and URP Instructions:
1. Import asset to an existing HDRP/URP project.
2. Import HouseFurniture_HDRP/URP.unitypackage file from Assets -> Import Package -> Custom Package. Unitypackage file Found from HDRP/URP folder.
3. Replace all files.
4. Shaders and materials should update to HDRP/URP.


Kitchen cabinets will snap to 0.1 grid.
Use Unity's ProGrids or Ctrl + L to open included AutoSnap script.
